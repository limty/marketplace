# The Marketplace
An App to buy or sell anything, in any condition.

## Technologies used
Rails, React JS, webpacker, Ruby, PostgreSQL.

## Installation instructions
```bash
bundle install
```
Then
```bash
rake db:create
```
and
```bash
rake db:migrate
```
